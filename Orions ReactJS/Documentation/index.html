Installation Everything you need to setup your react app quickly.

Step 1

Installing Node and NPM

You need to make sure that Node and NPM are installed on your system. If they are not installed then you can download them here

Step 2

Installing node modules

To install the node modules that the template is depended on, you need to open the command line terminal in the folder orions-react-demo-1, orions-react-demo-2 or orions-react-demo-3 and type the following command. This may take some time. 

npm install
Note: 'npm' command should be in the PATH environment variable to be accessible.

Step 3

Starting React App

To start the app, your current working directory should be same as in the previous step. You need to enter the following command in the command line terminal. After a few seconds, your react app will open in the default browser. 

npm run start
Note: 'npm' command should be in the PATH environment variable to be accessible.

Step 4

Deploying React App

To deploy the app, your current working directory should be same as in the previous step. You need to enter the following command in the command line terminal. After a few seconds, a folder named 'build' will appear in present working directory, which you can upload to your hosting server, and in four simple steps, your react app is visible to the world. 

npm run build
Note: 'npm' command should be in the PATH environment variable to be accessible.
